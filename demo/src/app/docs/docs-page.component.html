This is a demo of the iOS26 theme for Ionic Framework. Just import the CSS and start using it immediately.

% npm install @rdlabo/ionic-theme-ios26 --save

For detailed usage instructions, please refer to the README in the GitHub repository.

Special Coding

Additional work is required when using these iOS26 designs.

ion-button[fill=solid][type=submit]

Set color-brightness as a CSS variable. If using a `primary` button, the following is required. Please set your own color code.

--ion-color-primary-brightness: #96FEFF;
